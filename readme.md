# SYS-PORTFOLIO

Persönliches Portfolio eines angehenden  
**Fachinformatikers für Systemintegration**

---

## Ziel des Projekts

Dieses Projekt dient als Lern- und Dokumentationsplattform
während meiner Umschulung zum Fachinformatiker für
Systemintegration.

Der Fokus liegt auf:
- sauberer Struktur
- verständlichem Aufbau
- wartbarer Technik
- bewussten technischen Entscheidungen

---

## Projektbestandteile

### Portfolio
- Vorstellung meiner Person und Motivation
- Darstellung meiner Arbeitsweise
- Übersicht über Kompetenzen im Aufbau
- Thematisierung von Verantwortung & Cyber­sicherheit

### Lernjournal (Blog)
- Dokumentation des Lernfortschritts
- Reflexion technischer Entscheidungen
- Nachvollziehbare Entwicklungsschritte
- Keine Marketing-Texte, sondern technische Einordnung

### Digitale Assistentin
- Reagiert auf Navigation
- Begrüßt abhängig von der Tageszeit
- Startet bewusst stumm
- Keine externe API, kein Tracking
- Umsetzung ausschließlich mit JavaScript

---

## Technische Umsetzung

- HTML5
- CSS3 (Grid, Responsive Layout)
- JavaScript (ohne Frameworks)
- Keine externen Abhängigkeiten
- DSGVO-freundlicher Aufbau
- Fokus auf geringe Angriffsfläche

---

## Motivation

Dieses Projekt soll zeigen, wie ich:
- Probleme strukturiert angehe
- technische Lösungen Schritt für Schritt entwickle
- dokumentiere statt nur umzusetzen
- Verantwortung für Sicherheit und Wartbarkeit übernehme
