Linux statt Wegwerfen – Ubuntu als Lernsystem für alte Hardware

Ziel

Ein älterer Laptop, der nicht mehr Windows-11-tauglich ist, sollte sinnvoll weiterverwendet werden. Ziel war ein stabiles Linux-System, das später von meiner Tochter (7) zum Lernen und Spielen genutzt werden kann.

Ausgangslage

  • Lenovo-Laptop mit Windows-10-Lizenz
  • Hardware technisch funktionsfähig
  • Kein Upgrade auf Windows 11 möglich
  • Entscheidung für Linux statt Neukauf

Installation

Ubuntu 24.04 LTS wurde offline per USB-Stick installiert. Die Installation erfolgte bewusst minimal, ohne zusätzliche Software oder Treiber.

Erster Start & Systemprüfung

Nach dem ersten Start wurde der Systemzustand überprüft, einschließlich Version, Hardware und Grundkonfiguration.

Treiber & Secure Boot (WLAN-Problem)

Der Laptop nutzt einen Broadcom BCM43142 WLAN-Chip, der unter Linux einen proprietären Treiber benötigt. Da Secure Boot aktiv war, musste der Treiber über das sogenannte MOK-Verfahren autorisiert werden.

Nach der Autorisierung und einem Neustart war WLAN verfügbar und das System vollständig einsatzbereit.

Ergebnis

  • Ubuntu 24.04 LTS läuft stabil
  • WLAN funktioniert zuverlässig
  • System ist alltagstauglich
  • Kinderkonto ohne Adminrechte eingerichtet

Lernpunkte

  • Analyse von Hardware und Treibern unter Linux
  • Umgang mit Secure Boot und MOK
  • Bewusste Systementscheidungen
  • Nutzerorientierte Einrichtung für Kinder

Fazit

Linux ist eine sehr gute Möglichkeit, ältere Hardware sinnvoll weiterzuverwenden. Ubuntu bietet eine stabile, sichere und kostenlose Basis für Lern- und Alltagszwecke.
