Linux Ubuntu – Entwicklungsumgebung

Umstellung eines älteren Laptops auf Linux als stabiles Hauptsystem

Portfolio › Projekte › Linux Ubuntu
✅ Status: Abgeschlossen · Linux / Systemintegration

Projektziel

Ziel dieses Projekts war es, einen älteren Laptop, der nicht mehr Windows-11-tauglich ist, sinnvoll weiterzuverwenden. Statt eines Neukaufs wurde bewusst Linux als Hauptbetriebssystem gewählt.

Der Laptop soll zukünftig als stabiles Lern- und Spielgerät für meine Tochter (7) dienen und gleichzeitig als dokumentiertes Praxisprojekt im Rahmen meiner Umschulung genutzt werden.

Ausgangslage

  • Älterer Lenovo-Laptop
  • Windows 10 vorhanden, Upgrade auf Windows 11 nicht möglich
  • Hardware funktionstüchtig, aber nicht mehr zeitgemäß
  • Ziel: Wartbares, sicheres und alltagstaugliches System

Umsetzung

Installation

Ubuntu 24.04 LTS wurde offline per USB-Stick installiert. Die Installation erfolgte bewusst minimal, ohne zusätzliche Software oder automatische Treiberinstallation.

Treiber & Secure Boot

Der verbaute Broadcom BCM43142 WLAN-Chip benötigte einen proprietären Treiber. Da Secure Boot aktiv war, musste dieser über das MOK-Verfahren autorisiert werden.

Nach erfolgreicher Autorisierung und einem Neustart funktionierte die WLAN-Verbindung zuverlässig.

Benutzer- & Rechtekonzept

Es wurden getrennte Benutzerkonten eingerichtet. Für meine Tochter wurde ein Standardkonto ohne Administratorrechte erstellt, inklusive automatischer Anmeldung und vereinfachter Oberfläche.

Ergebnis

  • Ubuntu 24.04 LTS läuft stabil und performant
  • WLAN und Systemupdates funktionieren zuverlässig
  • Klare Trennung von Administrator- und Kinderkonto
  • Alltagstaugliches System für Lernen und Freizeit

Lernpunkte

  • Analyse von Hardware und Treiberproblemen unter Linux
  • Umgang mit Secure Boot und MOK
  • Bewusste, dokumentierte Systementscheidungen
  • Nutzerorientierte Einrichtung von IT-Systemen
