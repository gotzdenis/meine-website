HOMELAB

SYS-HOMELAB – Private Linux-Infrastruktur

Ziel

Ziel des SYS-HOMELAB-Projekts ist der Aufbau einer privaten Linux-Infrastruktur als Lern-, Test- und Dokumentationsumgebung für die FISI-Umschulung und zukünftige Infrastrukturprojekte.

Das Projekt dient als praktische Umgebung für Linux, Netzwerke, Serveradministration, Docker, Monitoring, Backup-Konzepte und IT-Sicherheit.

Aktueller Projektstand

  • Ubuntu Server 24.04 LTS installiert
  • Hostname „sys-homelab“ eingerichtet
  • System vollständig aktualisiert
  • OpenSSH erfolgreich eingerichtet
  • Erster SSH-Zugriff von Windows 11 durchgeführt
  • DHCP-Reservierung im TP-Link Router eingerichtet
  • Feste IP-Adresse vergeben
  • Headless-Betrieb erfolgreich getestet

Infrastruktur

  • Ubuntu Server 24.04 LTS
  • OpenSSH
  • Windows 11 SSH Client
  • DHCP-Reservierung
  • Lokale Netzwerkverwaltung

Erfolgreich abgeschlossen

Mit dem ersten betriebsbereiten Ubuntu-Server wurde die technische Grundlage des Homelabs geschaffen.

Der Server kann vollständig über SSH verwaltet werden und benötigt für den täglichen Betrieb keinen Monitor oder eine Tastatur.

Geplante Erweiterungen

  • Docker
  • Samba-Freigaben
  • Backup-Konzept
  • Monitoring
  • Grafana Dashboard
  • System-Härtung
  • Netzwerkdienste

Status

Der erste Meilenstein wurde erfolgreich abgeschlossen. Das Projekt befindet sich aktuell in der Aufbauphase und wird parallel zur FISI-Umschulung weiterentwickelt.
