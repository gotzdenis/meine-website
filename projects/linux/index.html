LINUX

Linux Ubuntu – Lern- & Entwicklungssystem

Ziel

Ziel dieses Projekts war die sinnvolle Weiterverwendung eines älteren Laptops, der die Anforderungen für Windows 11 nicht mehr erfüllte.

Statt ungenutzter Alt-Hardware entstand daraus ein stabiles Linux-System für Lernen, Entwicklung und praktische Systemerfahrung.

Ausgangslage

Die vorhandene Hardware war technisch weiterhin funktionstüchtig, jedoch nicht sinnvoll für moderne Windows-Upgrades geeignet.

Ziel war daher eine vollständige Umstellung auf Ubuntu Linux als nachhaltige Alternative zum Neukauf.

Technische Umsetzung

  • Installation Ubuntu 24.04 LTS
  • Systemanalyse und Hardwareprüfung
  • Broadcom BCM43142 WLAN Treiber Einrichtung
  • Secure Boot / MOK Autorisierung
  • Benutzerverwaltung und Grundkonfiguration

Technische Herausforderung

Eine besondere Herausforderung war der verbaute Broadcom WLAN-Chip, der unter Linux proprietäre Treiber benötigt.

Aufgrund aktivem Secure Boot musste die Treiberinstallation zusätzlich über das MOK-Verfahren autorisiert werden, bevor WLAN vollständig funktionierte.

Projektwert

Dieses Projekt war nicht nur eine technische Übung, sondern eine praktische Lösung zur sinnvollen Weiterverwendung vorhandener Hardware.

Perspektivisch kann das Gerät später innerhalb der Familie als Lern- und Alltagsgerät weiter genutzt werden.

Lernpunkte

  • Linux Installation & Systemeinrichtung
  • Hardwareanalyse unter Linux
  • Treiberprobleme systematisch lösen
  • Secure Boot und MOK Verständnis
  • Benutzer- und Rechteverwaltung
