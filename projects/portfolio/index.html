PORTFOLIO

SYS-PORTFOLIO – Technische Lernplattform & Portfolio

Ziel

Ziel dieses Projekts ist der Aufbau einer persönlichen technischen Dokumentationsplattform als Kombination aus Portfolio, Lernjournal und Projektübersicht während meiner Umschulung zum Fachinformatiker für Systemintegration.

Die Website dient nicht nur als Präsentation, sondern als echte technische Entwicklungsumgebung zur Dokumentation meiner Lernfortschritte, Infrastruktur-Projekte und praktischen Umsetzungen.

Warum SYS-PORTFOLIO?

Klassische Portfolio-Websites zeigen meist nur fertige Ergebnisse. Dieses Projekt verfolgt bewusst einen anderen Ansatz:

Nicht nur Ergebnisse dokumentieren — sondern den technischen Weg dorthin.

Dadurch entsteht eine nachvollziehbare Lernplattform, die sowohl technische Entwicklung als auch praktische Projekte sichtbar macht.

Technische Architektur

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • Statische Seitenstruktur
  • GitHub Pages Deployment
  • Lokale Browserfunktionen ohne externe APIs

Projektstruktur

Die Plattform wurde modular aufgebaut:

  • Portfolio als Hauptübersicht
  • Blog als technisches Lernjournal
  • Projektseiten für reale Infrastruktur-Projekte
  • Eigene Dokumentation pro Themenbereich

Aktueller Stand

SYS-PORTFOLIO verfügt aktuell über:

  • Einheitliches Cyber / Terminal Design
  • Strukturierte Blog-Dokumentation
  • Projektübersicht mit eigener Navigation
  • Modulare Projektordner
  • Saubere GitHub Pages Struktur
  • Responsive Darstellung

Geplante Erweiterungen

  • Homelab Infrastruktur Dokumentation
  • Praktische Netzwerk-Setups
  • Security Lernumgebung Ausbau
  • Linux Server Projekte
  • Weitere technische Lernmodule

Fazit

SYS-PORTFOLIO ist kein klassisches Showcase-Projekt, sondern eine aktiv genutzte technische Lern- und Entwicklungsplattform.

Das Projekt wächst parallel zur Umschulung und dokumentiert reale technische Entwicklung statt nur Endergebnisse.
