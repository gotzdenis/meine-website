SECURITY

Security Lab – Geplante Lernumgebung

Ziel

Ziel dieses Projekts ist der schrittweise Aufbau einer praktischen Lernumgebung für IT-Sicherheit, Systemhärtung und sichere Infrastruktur.

Warum ein Security Lab?

IT-Sicherheit ist ein zentraler Bestandteil moderner IT-Infrastruktur. Theoretisches Wissen allein reicht dabei langfristig nicht aus.

Dieses Projekt soll eine praktische Umgebung schaffen, in der Sicherheitsmaßnahmen getestet, Systeme abgesichert und reale Konzepte nachvollzogen werden können.

Geplante Inhalte

  • Linux System Härtung
  • Firewall Konzepte
  • SSH Absicherung
  • Benutzer- und Rechteverwaltung
  • Log Analyse & Systemüberwachung
  • Backup Strategien
  • Grundlagen Netzwerk-Security

Geplanter Stack

Linux Systeme, SSH, Firewall-Konfiguration, Benutzer- und Rechteverwaltung, Netzwerkanalyse und Sicherheitsmaßnahmen.

Status

Projekt aktuell in Planung.

Die praktische Umsetzung erfolgt schrittweise parallel zum Homelab und den fachlichen Lerninhalten der Umschulung.
