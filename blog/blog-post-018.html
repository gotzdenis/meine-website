SYS-Garden v1.0

Ziel

SYS-Garden ist eine digitale Gartenverwaltung zur Planung, Dokumentation und Organisation eines privaten Nutzgartens.

Ziel des Projekts ist die zentrale Verwaltung von Beeten, Gewächshausflächen, Pflanzen, Mischkulturen, Terminen und Notizen innerhalb einer eigenen Webanwendung.

Technische Umsetzung

  • HTML5
  • CSS3
  • JavaScript
  • LocalStorage
  • Git
  • GitHub
  • GitHub Pages

Aktueller Stand

  • Dashboard
  • Beetverwaltung
  • Gewächshaus
  • Pflanzenverwaltung
  • Obstverwaltung
  • Mischkultur Übersicht
  • Kalender
  • Notizen

Datenspeicherung

Die Anwendung nutzt aktuell LocalStorage. Dadurch besitzt jeder Besucher seine eigene lokale Gartenverwaltung innerhalb des Browsers.

Es werden keine Daten an externe Server übertragen und keine persönlichen Informationen gespeichert.
