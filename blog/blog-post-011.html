SYS-PORTFOLIO v2 – Technischer Umbau

Ziel

Die Portfolio-Website wurde strukturell überarbeitet, um eine klarere Trennung zwischen Portfolio, Projekten und Blog zu erreichen.

Änderungen

  • Einführung eines Terminal-Interfaces
  • Integration einer lokalen Assistentin
  • Verbesserte Navigation
  • Blogstruktur mit chronologischen Einträgen
  • Modularer Seitenaufbau

Technische Umsetzung

  • HTML5
  • CSS Grid Layout
  • JavaScript ohne Frameworks
  • keine externen APIs

Ergebnis

Die Website fungiert jetzt als persönliche Dokumentationsplattform während der Umschulung zum Fachinformatiker für Systemintegration.

Fazit

Durch den Umbau zu Version 2 wurde die Website strukturierter und modularer aufgebaut.

Das Portfolio dient nun nicht nur als Präsentation, sondern auch als technisches Lernjournal während der Umschulung.
