LINUX / SYSTEMINTEGRATION

Linux statt Wegwerfen – Ubuntu als Lernsystem für alte Hardware

Ziel

Ein älterer Laptop, der nicht mehr Windows-11-tauglich ist, sollte sinnvoll weiterverwendet werden.

Ziel war ein stabiles Linux-System, das später von meiner Tochter zum Lernen und Spielen genutzt werden kann.

Ausgangslage

  • Lenovo Laptop mit Windows-10-Lizenz
  • Hardware technisch weiterhin funktionsfähig
  • Kein Upgrade auf Windows 11 möglich
  • Bewusste Entscheidung gegen Neukauf

Installation

Ubuntu 24.04 LTS wurde per USB-Stick installiert.

Die Installation erfolgte bewusst minimal, ohne unnötige Zusatzsoftware, um ein sauberes und stabiles Basissystem aufzubauen.

Erster Start & Systemprüfung

Nach dem ersten Start wurde das System vollständig geprüft.

Dazu gehörten Versionskontrolle, Hardwareerkennung und die Überprüfung der grundlegenden Funktionalität.

Treiber & Secure Boot

Der Laptop nutzt einen Broadcom BCM43142 WLAN-Chip, der unter Linux einen proprietären Treiber benötigt.

Da Secure Boot aktiv war, musste der Treiber über das MOK-Verfahren autorisiert werden.

Nach der Freigabe und einem Neustart funktionierte WLAN zuverlässig und das System war vollständig einsatzbereit.

Ergebnis

  • Ubuntu 24.04 LTS läuft stabil
  • WLAN funktioniert zuverlässig
  • System ist alltagstauglich
  • Kinderkonto ohne Administratorrechte eingerichtet

Lernpunkte

  • Analyse von Hardware und Treibern unter Linux
  • Umgang mit Secure Boot und MOK
  • Bewusste Systementscheidungen treffen
  • Nutzerorientierte Einrichtung für reale Anwendungsfälle

Fazit

Linux ist eine sehr gute Möglichkeit, ältere Hardware sinnvoll weiterzuverwenden.

Ubuntu bietet eine stabile, sichere und kostenlose Basis für Lern- und Alltagszwecke.
